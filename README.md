Overview

This project is a React-based Development Books Store Kata where users can:

View a list of books

Add or remove books from the cart

View cart details

Automatically group books into unique sets

Apply discounts based on book sets

View final pricing details

The backend is mocked using json-server.

------------------------------------------------------------------------------------

Tech Stack

React.js

Context API + useReducer

Vitest (testing)

json-server (mock backend)

Tailwind CSS (styling)

-------------------------------------------------------------------------------------

Getting Started

Checkout the source code.

Open the source code directory in command line interface(CLI) and run npm install. This will add all the dependencies mentioned in the package.json file.

Run npm test for launching the test runner in the interactive watch mode.

Run npm run app for launching the application with json-server

Run npm run build for building the app for production to the build folder.
